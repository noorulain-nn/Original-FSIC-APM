**Abstract:**
Few-shot image classification remains difficult because models must adapt to new classes with only a small amount of labeled data. While prior work has largely focused on improving feature extractors, the role of the classification head in this setting has received far less attention. In this work, we address this gap by introducing Adaptive Prototype Memory (APM)—a non-parametric, cosine-similarity-based head that incrementally updates class prototypes and by comparing it directly with a conventional parametric softmax layer under a unified few-shot protocol. Two mechanisms enable APM adaptation: (i) a similarity-based update rule, where the update strength depends on feature–prototype closeness, and (ii) incremental updates, which refine prototypes gradually as new samples arrive. In our training setup, only the last block of the pretrained backbone is fine-tuned, while APM serves as the classification head. We evaluate this design across five benchmarks (CIFAR-100, CUB-200, Caltech-101, Omniglot, EuroSat) using diverse backbones including ResNet, DenseNet, SqueezeNet, VGG and Inception. This setup achieves 8–54 % point higher accuracy than models using a conventional softmax head, across these backbones. These findings show that when the backbone already yields discriminative embeddings, the challenge of few-shot adaptation shifts from feature learning to feature utilization, where APM consistently outperforms softmax layers.
